Зачем тебе свой сервер

Год я собирала автоматизацию из готовых сервисов: ChatGPT, n8n, боты, плагины. Половина не принимала российскую карту, половина блокировала по стране, остальное стоило как машина в год.

Решение простое: отдельный «компьютер в интернете», работает 24/7, принадлежит тебе. Называется VPS, аренда от 500₽ в месяц. На нём живёт Клод и знает твой бизнес, тон, процессы.

К нему подключается всё: телеграм-боты, таблицы, базы, Google Drive, Figma, сайт. Клод становится сотрудником, которого нельзя переманить и нельзя отключить извне.

Этот гайд про то, как поставить такого Клода. За один вечер. Без программиста.

С нейронками что-то создать уже не проблема. Главное понять, как оптимально решить свой запрос.

Я помогу.

Что понадобится

  1. Карта
    Российские работают на Beget и Timeweb. Для Claude Max нужна иностранная виртуалка, проще всего у бота «Плати по миру». Бери карту страны без санкций, желательно той же, что у твоего VPN. При оплате Anthropic VPN должен быть статичным, иначе привязка отвалится (шаг 06).
    «Плати по миру» →
  2. $20-$200 в месяц на Claude
    Начни с младшего тарифа, апгрейд в два клика. Эта подписка заменяет API-ключи, ботов и половину плагинов.
  3. 500-1000 ₽ в месяц на сервер
    Первый месяц минимум, потом подрастёшь по необходимости.
  4. Один вечер
    Реально 2-3 часа. Второй раз привычно, третий, за 30 минут.
  5. Любой ноутбук
    Mac, Windows, Chromebook. Через браузер и одну программу.

Программисты не нужны. Знания кода не нужны. Серьёзно.

01
Шаг 01

Купить сервер

Вика подсказка
Я брала Hostinger по реф-ссылке, −20% от цены. KVM 2, 4 ядра, RAM 4 GB хватает на всё.

Идёшь на хостинг и оформляешь VPS. Beget, русский интерфейс, российские карты. Для иностранной карты и сервера за границей, Hostinger или DigitalOcean.

Hostinger · −20% → Beget VPS → DigitalOcean →

Реферальная ссылка Hostinger, −20% на первый платёж.

На Beget:

  1. Открой beget.com/ru/vps.
  2. Тариф VPS Start или Standard (500-900 ₽/мес). Минимум 2 ГБ RAM, 2 ядра, иначе Клод задохнётся.
  3. ОС: Ubuntu 22.04. Именно она, не Debian.
  4. Пароль root, придумай и запиши. Потеряешь, пересоздавать сервер с нуля.
  5. Оплати картой. На выходе: IP (например 185.12.34.56), логин root и пароль.

Всё. У тебя теперь компьютер в интернете. Он работает прямо сейчас.

02
Шаг 02

Подключиться через VS Code

Вика подсказка
Кажется страшно, на деле 5 минут. Если застрял, пиши в личку, подскажу.

Можно ходить в сервер через голый терминал, но это страшно с первого раза. Я хожу через Visual Studio Code: бесплатный редактор Microsoft. Удобный интерфейс, файлы, встроенный терминал. И главное: Клод работает внутри сервера, у него нет доступа к твоему маку.

  1. Скачай VS Code с code.visualstudio.com, поставь как обычную программу.
  2. Открой VS Code. В Extensions поставь плагин Remote - SSH от Microsoft (синяя галочка).
  3. Там же, Claude Code от Anthropic. Официальный плагин, добавит кнопку запуска Клода.
  4. Открой Command Palette: ⌘ Shift P (Mac) или Ctrl Shift P (Win).
  5. Напечатай Remote-SSH: Connect to Host → Add New SSH Host.
  6. Вставь ssh root@185.12.34.56 (свой IP). Enter. Выбери ~/.ssh/config. Enter.
  7. Снова Command Palette → Remote-SSH: Connect to Host → выбери IP. Откроется новое окно. Linux? Да. Пароль, root-пароль с Beget.
  8. Готово. В нижнем углу, зелёная плашка SSH: 185.12.34.56. Ты внутри сервера.

Первый коннект, минута. Все следующие, 2 секунды, один клик.

03
Шаг 03

Запустить Клода внутри сервера

Вика подсказка
Когда Клод запустится, попроси: «представься и покажи что умеешь». Будет первым другом.

Магия.

Открой встроенный терминал: Terminal → New Terminal или Ctrl ` (обратный апостроф). Внизу появится командная строка, она работает на сервере.

Введи и нажми Enter:

curl -fsSL https://claude.ai/install.sh | bash

1-2 минуты установка. Жди, пока курсор замигает пустой.

Запускай:

claude

При первом запуске спросит как авторизоваться. Выбери Claude Max subscription (не API, Max дешевле и мощнее). Клод покажет ссылку, открой в браузере, залогинься в Anthropic, подтверди доступ.

Вернись в VS Code, Клод напишет Authenticated. Он твой.

Что получилось
ИИ живёт на твоём сервере. Ты смотришь на него через VS Code, твой мак его не видит. Как отдельная рабочая комната, туда заходишь работать, домашнее остаётся дома.
04
Шаг 04

Сказать Клоду «сделай себя удобным»

Вика подсказка
Это магия. После этого шага Клод сам адаптируется под тебя, папки, конфиги, правила. Просто говори словами.

Клод запущен, но пока пустой, как новый ноут без настроек. Надо объяснить кто ты и как работаешь. У программистов это занимает дни. Здесь, 5 минут и словами.

Не печатай, наговаривай голосом. На айфоне встроенная диктовка отлично понимает длинные мысли. На маке поставь приложение со словом Whisper в названии. Через голос мысль льётся естественно.

Промпт
Привет. Меня зовут [имя]. Я [кто, блогер, маркетолог, директор агентства]. У меня свой VPS, ты только что запустился. Настрой здесь всё для комфортной работы: базовая структура папок (заметки, проекты, база знаний, процессы), нужные пакеты, git, CLAUDE.md с моими правилами. На каждом шаге объясняй что делаешь.

Enter. Клод начинает сам выполнять команды: папки, пакеты, конфиги, проверки. На каждом шаге показывает «сейчас делаю это», «нужно для того», «готово».

Ты смотришь и соглашаешься. На каждое предложение он спрашивает подтверждения, жмёшь Enter, продолжает.

Первый раз, как фокус. На третий перестаёшь следить и идёшь пить чай.

05
Шаг 05

Первый скилл

Вика подсказка
Скилл, это ярлык. Один раз написал, один раз показал, дальше зовёшь по имени. Начни с чего-то, что делаешь каждую неделю.

Скилл, текстовый файл с инструкцией для Клода. Как служебка для сотрудника.

Например, ты часто пишешь посты в TG. Каждый раз объясняешь «тон», «структуру», «что не любишь». Это повторяется, идеальный кандидат на скилл.

Промпт
Создай мой первый скилл, для постов в TG-канал. Сохрани в .claude/skills/posts.skill.md. Сначала задай 10 вопросов про мой тон, структуру, запреты, примеры что нравится. Из ответов собери скилл по правильной структуре.

Клод задаст вопросы. Отвечаешь в свободной форме. Он превращает ответы в структурированный файл.

Теперь когда попросишь пост, Клод прочитает скилл и сработает по твоим правилам. Разница как между стажёром и сеньором.

В пакете к этой статье, 4 готовых скилла. Посмотри как примеры, укради структуру, перепиши под себя. Этому учит скилл skill-builder, он тоже в пакете.

06
Шаг 06 · бонус

Свой VPN на этом же сервере через Amnezia

Вика подсказка
Если статичный IP для Anthropic надоел через сторонний VPN, этот шаг для тебя. Один сервер на всё: и Клод, и VPN.

У тебя уже компьютер в интернете, почему бы не сделать его личным VPN. Без подписок, без «недоступно из вашей страны», без отвалов платных. И заодно статичный IP для Anthropic, привязка карты не отвалится.

Я пользуюсь AmneziaVPN: обходит блокировки в России, не требует ручной возни с конфигами. Серверную часть ставит сам десктопный клиент.

Скачать Amnezia → Гайд установки →
  1. Поставь Amnezia на мак или винду.
  2. В клиенте: «подключить свой сервер» → IP с Beget/Hostinger, логин root, пароль.
  3. Протокол AmneziaWG (WireGuard с обфускацией, главная фишка Amnezia). Установка 2-3 минуты, клиент всё ставит сам.
  4. Готово: значок «защита включена», трафик идёт через сервер, IP сервера = твой статичный IP.
  5. Для телефона: AmneziaVPN из App Store / Google Play. Расшарь QR из десктопного клиента, конфиг импортируется.
Итог
Один сервер закрывает обе задачи: AI-офис с Клодом и личный VPN на все устройства. Поедешь за границу, VPN с собой. Перед оплатой Anthropic убедись: Amnezia включена, страна сервера = страна карты от «Плати по миру».

Если что-то пошло не так

SSH не пускает, Permission denied. Неправильный пароль или IP. Пароль вводится «вслепую», курсор не двигается. Попробуй ещё раз. Если точно правильно, проверь IP в команде ssh root@IP.
Команды падают с ошибкой. Скопируй текст ошибки, вставь Клоду со словами «вот что сломалось, почини». Клод прочитает и либо починит, либо объяснит.
Клод не запускается, command not found. Установка не прошла до конца. Запусти npm install -g @anthropic-ai/claude-code ещё раз. Проверь интернет: ping google.com (Ctrl+C прервать).

Что дальше

У тебя сервер, Клод и базовая настройка. Это как купить машину, пора поехать. Дальше в порядке ценности:

  1. Создать свои скиллы
    По одному процессу за раз. Не описывай всё сразу, путь в хаос. Один скилл → проверь на реальной задаче → доработай. За месяц 10-15 штук.
  2. Выгрузить свою экспертизу
    Знания, что нарабатывались годами, переговоры, тексты, продажи. Пока в голове, ты единственный источник. Выгрузи в текст, зашей в Клода, появится «второй ты». Скилл expertise-extractor в пакете.
  3. Подключить свои данные
    Drive, Календарь, заметки, CRM. Через MCP-серверы, одной командой. Клод начнёт работать с реальными данными.
  4. Бот-интерфейс
    Чтобы не лезть в терминал, телеграм-бот с голосовыми. Кинул мысль на бегу, получил черновик.
